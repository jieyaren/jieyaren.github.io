压缩图片 

- jpg by `jpegtran`

```bash
apt install libjpeg-progs
jpegtran -optimize image-20200402171439048.jpg
```



- png by  `optipng`

```bash
apt install optipng
optipng -o3 image-20200402172644242.png
```

o1 ~ o7 七个等级压缩
